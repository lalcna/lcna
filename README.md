# Salemthar Information System (CIS)

The **Salemthar Information System (CIS)** is a centralized digital platform designed to modernize and streamline communication within the TKP Salem Unit. The system replaces scattered communication channels like paper notices, local church newspapers, and verbal announcements by providing a unified digital hub that stores, displays, and archives sermons, articles, notices, reports, and posts.

This project is built with **PHP**, **MySQL**, **HTML/CSS**, and **Vanilla JavaScript**, designed for simplicity, speed, and long-term maintainability.

----

 Features:

-  **Daily content filtering** (View all items for a selected date)
-  **Articles, Posts, Notices** (with expand/collapse view)
-  **Clean bulletin-style layout**
-  **All Articles Archive page**
-  **Admin-controlled PDF download lock**
-  **Fully responsive UI**
-  Custom CSS with modern design
-  Lightweight and fast (no frameworks needed)
